How to Use Githup Page to Create Blog

Posted on 2020-07-13 Edited on 2020-09-15

Quick Start

Hexo Init

Create A Repository

must create before hexo deploy【must be {user}.github.io or css will be not found】

Hexo Deploy

vi _config.yaml

Githup Page

  1. goto Repositoty/Setting/Option/Githup Page
  2. choose your deploy brache
  3. when page done, you will see the visited url

Dns Setting

  1. Buy a domain in godaddy
  2. Add A Record in Dns Setting of your domain
  3. Goto Repositoty/Setting/Option/Githup Page. Update [Custom domain] to yours

Some Notes

  • when you deploy your blog, though the code is commited, but you need to wait a moment
  • if wait a moment not usefule, push a new commited
